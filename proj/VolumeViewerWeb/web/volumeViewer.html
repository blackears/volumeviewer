Running at the command line

Compiled jars

BodyViewer.jar

BodyViewerTest.jar

vecmath.jar

Third party projects

JOGL

vecmath

To run these files on the command line, download these jars and place them in the same directory. Also, copy the jogl.jar and gluegen-rt.jar from the JOGL distribution for your platform and place them in the same directory. Finally, copy the .dll or .so files from the JOGL distribution into the same directory (or set your system path so that they can be found).

To run on the command line:

java -jar BodyViewerTest.jar
